List group

List group are a flexible and powerful component for displaying a series of content. Modify and extend them to support just about any content within.

Basic example

The most basic list group is an unordered list with list items and the proper classes. Build upon it with the options that follow, or with own CSS as needed.

  • An item
  • A second item
  • A third item
  • A fourth item
  • And a fifth one
<ul class="list-group">
<li class="list-group-item">An item</li>
<li class="list-group-item">A second item</li>
<li class="list-group-item">A third item</li>
<li class="list-group-item">A fourth item</li>
<li class="list-group-item">And a fifth one</li>
</ul>
This is an attempt to recreate Bootstrap 5 official documentation page, please visit the site for more and updated details.

All the information given above is from the v5.0.0-beta2. please do remember that bootstrap releases new versions which mostly are similar with some updated changes of the older versions.
